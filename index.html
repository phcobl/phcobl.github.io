Laser Lane

A simple 2D game prototype made with Unity, in which a player controls a car driving on a road in a sidescroller fashion. The player can accellerate and decellerate as well as switch lanes to avoid other cars. Passing a car scores a point. Crashing removes all points.

Playable prototype on itch.io.

Project Haj

A 2D platformer made in the span of a weekend using Godot. Made for Godot Wild Jam - Wild and Proud Mini Jam. The game features a shark that can be controlled with jumps and can grab onto walls.

Playable prototype on itch.io.

Factory Game

Implementation of 2D factory game mechanics made to learn Godot syntax and GDscript.

Playable prototype on itch.io.

Skilltree Builder

An graphical interface to create skill-trees for a tabletop rpg. Made with Unity.

Playable prototype on itch.io.

VR Interface and Interaction

Interactive VR-environment with implementation of a set of mechanics. An audio interface lets players interact with the world through speech-recognition. The 3D-UI implements paradims of VR-GUI design to create an easy-to-use interface.

Demo video for Multimodal Interface available on YouTube.

Demo video for 3D-UI available on YouTube.

Gesture Recognition (ML)

Python implementation of a neural network. The network was trained on video data to recognize a variety of gestures in a live video feed. The input read from the gestures can be used to e.g. control presentation slides.

Demo video available on YouTube.
